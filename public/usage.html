The general format of the url is:
multilink.herokuapp.com?links={links}&mode=(manual|automatic)
Example:
multilink.herokuapp.com?links=www.google.com,www.yahoo.com,www.bing.com
You'll notice how mode is ommited from the example. When ommited it defaults to manual.
Here's another example, this time with mode set to automatic.
multilink.herokuapp.com?links=www.reddit.com,www.slashdot.org&mode=automatic
There are a few things to keep in mind when creating links:
  • If the protocol is not supplied (as in the example above) it will default to http.
  • javascript: links are not allowed and will just be ignored.
  • Currently only http(s), ftp and mailto protocols are supported.
